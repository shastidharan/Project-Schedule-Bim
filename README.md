# BIM Project Schedule Dashboard

A BIM project scheduling and tracking dashboard.

## Live Demo
Deployed on Vercel — see deployment URL.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript (standalone)
- **Hosting:** Vercel (free tier)
- **Backend:** Supabase (database, auth, APIs)

## Local Development
Just open `index.html` in any browser — no build step needed.

## Deployment
This repo is connected to Vercel. Every `git push` to `main` triggers an automatic redeploy.

## Environment Variables (set in Vercel dashboard)
```
VITE_SUPABASE_URL       = https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY  = your-publishable-key
```
